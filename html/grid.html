Grid

Grid Layout es un sistema de diseño que permite organizar elementos en filas y columnas, facilitando la creación de interfaces complejas y responsivas.

Contenedor de Grid: Es el elemento padre que establece un nuevo contexto de formato para los elementos hijos. Se activa al aplicar la propiedad display: grid en un elemento HTML.

Ítem de Grid: Son los elementos hijos dentro del contenedor de grid.

Celdas (Grid Cells): La unidad mínima de la cuadrícula, donde cada celda se forma por la intersección de filas y columnas.

Banda (Grid Track): Se refiere a las filas y columnas que componen la cuadrícula. Las bandas pueden tener tamaños fijos o flexibles, utilizando unidades como píxeles, porcentajes o la unidad fr (fracción).

Líneas (Grid Lines): Son las líneas que separan las celdas en la cuadrícula, tanto vertical como horizontalmente.

Área de Grid (Grid Area) Un Grid Area es un conjunto de celdas que forman un área rectangular dentro de la cuadrícula. Las áreas se definen mediante: Posicionamiento explícito: Se puede asignar un área a un ítem utilizando la propiedad grid-area, que puede referirse a una celda específica o a un grupo de celdas. Nombres de área: Utilizando la propiedad grid-template-areas, se pueden crear áreas con nombres descriptivos que facilitan la organización visual del layout.
