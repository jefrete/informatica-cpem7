Configuración de Visual Studio Code para Python

1. Descargar e instalar Visual Studio Code

  1. Ve al sitio oficial: Visual Studio Code.
  2. Descarga la versión correspondiente a tu sistema operativo.
  3. Instala VS Code siguiendo las instrucciones del instalador.

2. Instalar la Extensión de Python

  1. Abre Visual Studio Code.
  2. Haz clic en la pestaña de Extensiones (o presiona Ctrl + Shift + X).
  3. En la barra de búsqueda, escribe Python.
  4. Selecciona la extensión de Microsoft y haz clic en Instalar.
  5. Si el botón de ejecución (triángulo) no aparece en archivos Python, reinicia VS Code.

3. Descargar Carpeta de Google Drive y Abrir en VS Code

  1. Abre Google Drive y encuentra la carpeta deseada.
  2. Haz clic derecho y selecciona Descargar.
  3. Extrae el archivo ZIP en la ubicación que prefieras.
  4. Abre Visual Studio Code.
  5. Haz clic en Archivo > Abrir Carpeta y selecciona la carpeta extraída.
